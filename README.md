# 介绍
spring-boot & mybatis & sqlite

# 项目结构
```
├── README.md
├── pom.xml
├── src
│   ├── main
│   │   ├── java
│   │   │   └── cn.liq
│   │   │       └── blog
│   │   │           ├── BlogApplication.java
│   │   │           ├── entity
│   │   │           │   └── User.java
│   │   |   └── resources
│   │   |           └── application.yaml        
```
# 使用说明
1. 导入项目
2. 启动项目
3. 访问 http://localhost:8080

# 注意事项
1. 项目中使用了 lombok，所以需要安装 lombok 插件
2. 项目中使用了sqlite，所以需要安装 sqlite 插件
3. 项目中使用了 spring-boot-starter-web，所以需要安装 spring-boot-starter-web
